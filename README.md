## 使用語言
Python
## 開始遊戲
前置作業
``` Python
pip install pygame
```
啟動
``` 
python main.py
```
## 遊戲說明
  玩家需要操縱角色去躲避天空中落下的劍，遊戲中存在道具幫助玩家生存，注意別掉入海中會直接導致遊戲結束!
